二、功能模块需求​

（一）用户端功能（前端）​

1. 身份与会员体系​

注册登录：支持手机号验证码注册、微信 / 支付宝快捷注册；密码 / 验证码 / 第三方账号登录，7 天免密自动登录，跨设备登录状态同步​

会员管理：成长值累计（消费 1 元 = 1 成长值），等级划分（普通→白银→黄金→钻石），等级权益展示（折扣、运费券、专属客服）​

个人中心：基本信息编辑（昵称、头像、性别）、账号安全设置（密码修改、手机号绑定）、积分明细查询​

2. 商品浏览与互动​

商品展示：多图轮播（支持放大查看细节）、30s 内短视频展示、规格选择（尺码 / 颜色与库存联动）、搭配推荐（"常与该商品搭配的单品"）​

搜索体验：关键词联想、历史搜索标签化、搜索纠错（如 "血拼"→"雪纺"）、结果排序（综合 / 价格 / 销量）​

用户互动：图文评价（9 图 + 文字）、视频评价（15s 内）、问答社区（用户提问 / 商家回复）、用户实拍（购买者上传穿着效果）​

3. 交易流程​

购物车：商品暂存（未登录本地存储，登录后合并）、库存实时校验、失效商品标记（下架 / 库存不足）、批量结算​

结算中心：地址管理（新增 / 编辑 / 默认设置）、支付方式选择（微信 / 支付宝 / 银行卡 / 余额）、优惠券 / 积分抵扣、发票申请​

订单管理：订单状态跟踪（待付款 / 待发货 / 待收货 / 已完成）、物流轨迹可视化、订单操作（取消 / 确认收货 / 退款申请）​

4. 营销活动​

优惠券中心：优惠券领取 / 查看 / 使用、过期提醒​

活动参与：限时折扣、满减活动（满 200 减 30）、组合套餐（2 件 8 折）、分享得券（分享商品给好友双方得满 100 减 10 券）​

（二）管理端功能（后端 + 前端）​

1. 商品管理​

基础操作：商品新增 / 编辑 / 删除、批量上下架、Excel 导入导出、库存预警设置（低于 50 件提醒）​

内容管理：富文本详情编辑（支持视频 / 表格插入）、多图上传（主图 / 细节图 / 穿搭图）、SEO 配置（标题 / 关键词 / 描述）​

规格管理：规格模板设置（S/M/L 通用模板）、SKU 管理（尺码 / 颜色组合配置）​

2. 订单处理​

订单管理：多条件筛选（订单号 / 用户 ID / 时间 / 状态）、批量发货（导入物流单号）、异常订单标记（地址错误 / 支付超时）​

售后处理：退款申请审核、退货原因统计、售后进度跟踪​

数据统计：订单来源分析（PC/H5 / 小程序）、支付方式占比、销售额趋势​

3. 营销管理​

活动配置：限时折扣设置（支持 "前 100 名额外 9 折"）、满减规则配置、组合套餐创建​

优惠券管理：券类型设置（满减 / 折扣 / 无门槛）、领取限制（每人限领 2 张）、使用场景限定（全品类 / 指定分类）​

4. 数据分析​

核心看板：实时销售额（今日 / 昨日对比）、订单转化率、用户留存率（7 日 / 30 日）​

用户分析：用户画像（年龄 / 性别 / 地域分布）、消费偏好统计​

商品分析：热销 TOP10、滞销商品预警、生命周期分析​

（三）UI 设计需求​ (使用 Figma 设计一套完善的UI设计图 )

用户端 UI：简洁现代风格，主色调选用浅蓝 + 白色（体现服装清新感），响应式设计（适配 PC / 手机 / 平板），交互反馈（按钮动效 / 加载状态）​

管理端 UI：专业高效风格，采用左侧导航 + 右侧内容布局，数据可视化图表（折线图 / 柱状图 / 热力图），批量操作按钮突出显示​

设计规范：统一按钮样式、表单间距、图标库，确保页面一致性；关键操作（支付 / 提交）设置二次确认
